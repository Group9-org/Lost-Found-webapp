# CampusFind

CampusFind is a modern university lost and found platform designed to help students report, search, and recover lost items efficiently.

## Features

- Lost item reporting
- Smart search system
- Notifications
- Responsive UI
- Modern landing page
- Smooth animations

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- AOS Animation Library

## Team Project

Developed as a LUANAR computer Science group project.
